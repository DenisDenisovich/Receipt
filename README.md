- Clean Architecture
- MVVM
## Used technologies
- Kotlin
  - Coroutines
- [Android Jetpack](https://developer.android.com/jetpack)
  - ViewModel
  - LiveData
  - Room
  - CameraX
- [Koin](https://insert-koin.io/) - Service Locator
